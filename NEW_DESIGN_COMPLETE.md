## 🌈 配色系統

### 主色調（4 種霓虹色）
```css
--cyan-main: #00e5ff      /* 青色 - 70% 使用 */

--pink-main: #ff3d7f      /* 粉色 - 10% 使用 */
--purple-main: #a855f7    /* 紫色 - 5% 使用 */
```

### 使用分配
- **Header 標題光暈**: 青色
- **Navigation Active**: 青色 → 紫色漸變
- **Navigation 底線**: 青 → 粉 → 橘彩虹
- **背景光點**: 青 + 橘 + 粉混合
- **Section 標題**: 青色
- **按鈕**: 各種顏色
